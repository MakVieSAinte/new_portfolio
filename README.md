# Portfolio

Un portfolio minimaliste construit avec **Vue 3**, **TypeScript** et **Vite**.

## Stack

- Vue 3 + TypeScript
- Vite
- Tailwind CSS + Preline UI
- Vue Router
- Supabase + Hashnode GraphQL

## Démarrer

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Fonctionnalités

- Chargement ultrarapide avec Vite
- Thème clair/sombre
- Multilingue (FR/EN)
- Responsive design
- Accessible
- SEO optimisé
- Blog intégré
- Déployé sur Vercel
